# Flow JSON v7.3 — applied validation rules

Reverse-engineered from Meta's `WAFlowJSONValidatorChainV703`. Lists every validator the official Meta Flow Builder runs on a Flow JSON declaring `"version": "7.3"`, plus the full error-code dictionary.

> Source: `scripts/WAFlowJSONValidator_bundle.js`. Regenerate with `node scripts/extract_v703_rules.js`.

---

## Chain lineage

V703 inherits all validators from earlier chains. Newer chains add or replace rules; nothing is removed implicitly.

## How to use this reference

1. Run `node scripts/run_validator.js <flow.json>` — output includes an `errors` array.
2. Each error has a `code` — look it up in the table above to confirm what failed.
3. To dig deeper, search `scripts/WAFlowJSONValidator_bundle.js` for the validator name from the first table; its `validate()` function holds the exact rule.
