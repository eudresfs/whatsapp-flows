## Suporte (Support)

Se você estiver com dúvidas sobre o WhatsApp Flows, comece por aqui para obter ajuda. Se você quiser enviar um feedback, este também é o lugar certo.

> _(English gloss: If you have questions about WhatsApp Flows, start here for help. If you want to send feedback, this is also the right place.)_

---

### Suporte Direto (Direct Support)

Caso você seja um parceiro gerenciado, selecione _WABiz: Cloud API / WABiz: On Prem_ e, depois, selecione o tipo de solicitação como _WA Flows_.

> _(English gloss: If you are a managed partner, select **WABiz: Cloud API / WABiz: On Prem** and then select the request type as **WA Flows**.)_

---

### Comunidade de desenvolvedores (Developer Community)

Você pode verificar as [perguntas atuais para desenvolvedores aqui](https://developers.facebook.com/community?sort=trending&category=766772797555412&tag=870321727692773). Ao fazer uma pergunta, use _Categoria: WhatsApp Business API > Produto: WhatsApp Flows_.

> _(English gloss: You can check current developer questions [here](https://developers.facebook.com/community?sort=trending&category=766772797555412&tag=870321727692773). When asking a question, use **Category: WhatsApp Business API > Product: WhatsApp Flows**.)_
